# Review System

用复习项对抗记忆衰减和面试盲点。

## 建议复习间隔

- 第一次：当天
- 第二次：1 天后
- 第三次：3 天后
- 第四次：7 天后
- 后续：根据信心间隔 14 到 30 天

## 记忆状态

- `new`：刚创建，未演练
- `shaky`：只能看笔记回答
- `usable`：能回答，但不够精炼
- `strong`：压力下也能清楚回答
- `stale`：曾经熟练，但最近没复习

## 优先级规则

优先复习“简历价值高、攻击风险高、记忆强度低”的内容。
