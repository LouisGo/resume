TypeX PC 是面向高频 IM 协作场景的桌面端产品，核心复杂度集中在消息列表、消息渲染、Electron SDK Host、多端 SDK 对接和高可靠交互体验。随着大模型能力进入产品主流程，复杂客户端不再只是消息 UI 和业务入口，而需要承载实时通信、原生 SDK、消息领域模型和 AI 工作流。

- **IM Message List 视口运行时与滚动治理**：主导高频会话列表从组件级虚拟列表优化演进为消息锚点驱动的视口模型，覆盖动态高度、双向分页、定位跳转、follow-bottom 和会话切换恢复等场景；将滚动正确性从零散补丁收敛为可验证、可回滚、可持续对接 SDK 分页能力的底层方案。
- **Message Render 领域建模与插件化渲染**：核心负责 Message Render 建设，将 SDK Raw Message 转换为稳定 `MessageModel`，通过类型注册表、manifest、capability 和渲染插件拆分类型识别、数据清洗、布局决策与 UI 展示；以控制反转、selector hooks 和场景化 provider 建立渲染边界，使新增消息类型从改动主流程变为注册独立能力单元。
